Kỹ năng chuyên môn

Dưới đây là tổng hợp các kỹ năng chuyên môn và kỹ năng mềm mà tôi đã tích lũy được trong quá trình học tập và làm việc thực tế.

Lập trình Web (Front-end)

Thành thạo xây dựng giao diện website:

  • HTML5 / CSS3: Dựng layout chuẩn SEO, Semantic.
  • JavaScript (ES6+): Xử lý logic, DOM manipulation.
  • Responsive Design: Tối ưu hiển thị trên Mobile/Tablet.

Tư duy & Giải thuật

Nền tảng khoa học máy tính vững chắc:

  • Cấu trúc dữ liệu và giải thuật.
  • Lập trình hướng đối tượng (OOP).
  • Quy trình phát triển phần mềm (SDLC, Agile/Scrum cơ bản).

Cơ sở dữ liệu (Back-end Basic)

Khả năng thiết kế và truy vấn dữ liệu:

  • MySQL / SQL Server: Viết câu lệnh truy vấn, tạo bảng, quan hệ dữ liệu.
  • Kết nối CSDL với ứng dụng (PHP/C#).

Kỹ năng mềm & Công cụ

Hỗ trợ công việc hiệu quả:

  • Git/GitHub: Quản lý phiên bản mã nguồn.
  • Làm việc nhóm: Giao tiếp, thuyết trình đồ án.
  • Tiếng Anh: Đọc hiểu tài liệu kỹ thuật tốt.
